How to add background image in Android Studio

Step 1:


Open Android Studio and create a new project.

Step 2:


After the project is loaded, add an image in the Drawable folder.


Step 3:


Now open your layout file which is present in this path res -> layout -> activity_main.xml file

Step 4:


Now add below in code in your starting layout declaration. 

 
 android:background="@mipmap/ic_launcher
